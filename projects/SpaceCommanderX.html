Space Commander X

I developed a Nostalgic Asteroid-type game that is accessible on the site.

Game DevelopmentBlenderC#

Space Commander X is a game developed on Unity where you control a ship in which you are to use to defeat oncoming forces. Basic controls are used, WASD for example along with support for arrow keys as well. The game is never-ending, meaning that it will not stop until you as the player is defeated.

This project explained more about how the functions of C# work in Unity, along with pairing different platforms to create one project. It exercised the ability to produce a single aspect game and bring it to fruition with many of the tools at Unity’s disposal. I think with this, one would be able to create many other things aside from this. Here is an example of the code.
